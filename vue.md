# Vue声明式渲染
1. 声明式渲染
- 只需要声明在哪里where,做什么what,而无需关系如何实现how
2. 指令式
- 需要以具体代码表达在哪里where做什么what,如何实现how
### 声明式渲染理解
1. DOM状态只是数据状态的一个映射
2. 所有逻辑尽可能只是数据状态的一个映射
3. 当状态改变了，view会被框架自动更新到合理的状态

# Vue指令
1. v-bind   绑定属性
2. v-model  双向数据绑定
3. v-for    循环
4. v-if     移除添加
5. v-show   隐藏删除
6. v-text   跟{{}}类似
7. v-html   将数据内的标签解析成元素（注意，这个html必须是可靠的，如果是用户输入的，会被xss注入）
8. v-cloak  如果没有渲染完，就会被隐藏
9. v-once   只会渲染一次，以后再改就不会渲染，就变成静态页面，用于页面优化
10. v-on    事件

### 事件修饰符
> 方法只有纯粹的数据逻辑，而不是去处理DOM事件细节
@事件名.修饰符.修饰符=""

1. .stop    阻止事件冒泡
2. .prevent 阻止默认事件
3. .capture 事件捕获
4. .self    只能自身触发事件才能触发，不会受冒泡影响
5. .once    只触发一次
6. .enter/.tab/.delete/.esc/.space/.up/.down/.left./.down/.键值      键盘事件触发
